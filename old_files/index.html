Web Tetris

Github Repsitory

A version of the classic game Tetris built for the web using React/Redux, Node.js, and Express. A partner and I had a week to build this game; the live version only has basic functionality at the moment, but the development version currently supports multiplayer vs mode. In this mode a player can drop extra pieces into their opponents game upon a certain amount of line completions.
